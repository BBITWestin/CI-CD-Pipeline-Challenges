## Contributing

1. Fork the Project
2. Create your Feature Branch
   ```sh
   git checkout -b feature/AmazingFeature
   ```
3. Commit & Push your Changes
   ```sh
   git add . ; git commit -m "Some AmazingFeature" ; git push
   ```
4. Open a Pull Request
